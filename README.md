⚙️ Installation
# Clone repo
git clone https://github.com/Vikashsingh03/ai-interview-app.git

# Go to project
cd ai-interview-app

# Install frontend
cd frontend
npm install

# Install backend
cd ../backend
npm install
▶️ Run Project
# Backend
cd backend
npm run dev

# Frontend
cd frontend
npm run dev
🌐 Environment Variables

Create .env file in backend:

PORT=3000
MONGO_URI=your_mongodb_url
GEMINI_API_KEY=your_api_key
📦 Deployment

🚀 Frontend → Vercel / Netlify
🚀 Backend → Render / Railway
